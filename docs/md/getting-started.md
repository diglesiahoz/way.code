# Instalación

1. Clona proyecto e instala
```console
git clone https://github.com/diglesiahoz/way.code.git && \
cd way.code && \
make install && \
source ~/.bashrc && \
make up && \
make logs
```
2. Establece configuración inicial y ejecuta
```console
way core.init && \
source ~/.bashrc && \
way
```
